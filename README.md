# Automatic Staircase Light using Arduino

An automated lighting system designed for staircases using Arduino and motion sensors (PIR/Ultrasonic). The system detects human presence and automatically illuminates the stairs, saving energy and providing convenience.

## Features
- Automatic motion detection using sensors
- Smart LED lighting sequence for stairs
- Energy-efficient and automated control

## Hardware Components
- Arduino (Uno / Nano)
- Motion Sensors (PIR or Ultrasonic)
- LEDs / LED Strip
- Transistors / Relay Module
- Power Supply
